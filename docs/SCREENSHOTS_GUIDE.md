# MortgageUS — Screenshots Guide

## Required Screenshots

Google Play requires **2–8 screenshots** per device type.  
App Store requires screenshots for **iPhone 6.7"** and optionally larger sizes.

## Tools for Screenshots

- **Android**: Android Studio emulator → Extended Controls → Screenshot
- **iOS**: Xcode Simulator → File → Save Screen
- **Framing**: Use [MobileFrame.dev](https://mobileframe.dev) or Figma device frames
- **Text overlay**: Canva, Figma, or Adobe Express — add caption at bottom

---

## Tips

- Use a real loan amount (not $0 or $1) — looks professional
- Show results, not just the input form
- Dark mode screenshot adds value (shows you support it)
- Screenshot order on Play Store matters — put the most compelling first
